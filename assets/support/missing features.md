If global keybinds and possibly some more features aren't working for you, it's likely you installed the Equicord Desktop app, "Equibop".

Equibop is a standalone independent app that is missing some features from the regular Discord app.

If you need such features, stop using Equibop, and switch to the regular Discord app patched with Equicord. To do so, head over to <https://equicord.org/download> for Download options

This is well documented in the README, so please pay more attention in the future.
