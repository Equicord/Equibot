If the `Ctrl` + `Shift` + `I` keybind does not work, it is likely because a different program conflicts with the keybind. This is most commonly the AMD Radeon Software. If you have that, disable keybinds in it (see the video below).
[Visual demonstration](https://equicord.org/assets/support/console.mp4)
